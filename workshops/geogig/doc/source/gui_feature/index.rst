QGIS Sample Workflow 2
======================

This section provides another sample workflow using the QGIS GeoGig plugin. 
The example will introduce a workflow that would be suitable for GIS analysts adding new features to the **bikepdx** layer, sourced from fieldwork or digitization.  Our example will focus on a single data collector and an optional data manager.
